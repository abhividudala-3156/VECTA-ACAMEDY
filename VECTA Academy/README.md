# VECTA ACADEMY Sample Learning Website

This is a beginner-friendly sample frontend website for the VECTA ACADEMY, designed with a nature-inspired theme to showcase modern online courses. 
# contributor
V Abhinay Kumar

## Website Type
Educational / Online Courses Website

## Technologies Used
- HTML5
- CSS3
- JavaScript
- GitHub
- Netlify

## Pages Included
- Home
- About
- Courses
- Announcement
- Registration
- Contact 

## Features
- Responsive design for mobile, tablet and desktop
- SEO meta tags on key pages (title, description, keywords) 
- Image alt text and accessible course thumbnails
- Sticky navigation header with mobile menu toggle
- Course filter on the Courses page (All / Technology / Design / Business) using JavaScript 
- Weekly timetable section showing course schedule and instructors 
- Registration / enquiry form (optional) with basic JavaScript validation

## How to Run
Open `index.html` in any browser.

## Netlify Deployment Steps
1. Create a GitHub repository.
2. Upload all files from this folder (HTML, CSS, JS, images).
3. Login to Netlify.
4. Click "Add new site" > "Import an existing project".
5. Connect your GitHub repository.
6. Deploy the website.
